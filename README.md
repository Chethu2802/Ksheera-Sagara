# Ksheera Sagara (ಕ್ಷೀರ ಸಾಗರ)

Dairy farm management app built with Kotlin + Jetpack Compose.

## Features
- Bilingual login/signup (English + Kannada)
- Dashboard with profit/loss, monthly report
- Cow analysis & AI suggestions
- Year calendar to add income/expense on any date
- Download & share report

Open in **Android Studio Hedgehog or newer** and run.
